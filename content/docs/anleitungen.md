# Hygiene

## Installierte Programme verwalten / deinstallieren

### Windows 11/10

In Windows können alle installierten (nicht portabelen) Apps über die Einstellungen erreicht werden.

Erreichbar sind diese über Suche -> Einstellungen Einstellungen bzw. Windowstaste + I.
Von hieraus können dann über Apps -> Installierte Apps alle installierten Apps angzeigt werden.

Alternativ gibt es die Möglichkeit über folgenden Link zu diesen Einstellungen zu gelangen:
"ms-settings:appsfeatures" die [hierüber](ms-settings:appsfeatures) aufgerufen werden kann.

Über die drei Punkte an der rechten Seite des jeweiligen Eintrags können Anwendungen verschoben und deinstalliert werden.

### MacOS

In MachOS gibt es verschiedene Wege sich alle Apps anzeigen zu lassen: alle installierten Apps können über das Launchpad angzeigt werden.

Auch über Finder -> Applications / Programme können alle angzeigt werden. Über die verschiedenen Buttons weiter oben, können die verschiedenen ansichten durchgeschalten werden.

Typischerweise können Anwendungen deinnstalliert werden, indem Sie auf bzw. zu den Papierkorb gezogen werden. Mit dem löschen des Papierkorbs wird die Anwendung entgültig gelöscht. Jedoch bleiben hier meist noch Persönliche Daten und Einstellungen enthalten. Um diese zu löschen ist es meist nötig auf der Webseite des Entwicklers bzw. Herstellers zu prüfen wo und wie Daten abgelegt werden und wie diese entfernt werden können.
Manche Hersteller bieten außerdem einen zusätzlichen Uninstaller bereit. Dies ist bei komplexer Software wie der Adobe Cloud Platform der Fall. Diese Uninstaller können über die Herstellerseite heruntergeladen werden.
Zudem gibt es noch Apps die eine Option anbieten über die die Anwendung deinstalliert werden kann.

Die Option und der Uninstaller ist hier zu bevorzugen. Sofern man die Anwendung nicht mehr benötigt werden hierdurch in der Regel auch Einstellungen und Persönliche Daten entfernt, die bei der ersten Methode ohne manuelle Reinigung auf dem Gerät bleiben.

Unter MacOS ist es abhängig von der App, wie diese möglichst restlos entfernt werden kann.
Einige Apps fügen beim installieren auch eine separate App für das deinstallieren der App bereit. Einige Apps bieten auch eine Option bzw. Funkiton "Deinstallieren" über die Anwendungen vergleichbar zur vorhiergen Methode entfernt werden können. Falls diese beiden Optionen bei der zu deinstallierenden App der Fall verfügbar ist, ist dieser Weg den nun genannten zu bevorzugen, da hier Chaches bzw. systembezogene Daten zusätzlich entfernt werden, die andernfalls noch auf dem Mac gespeichert werden.
Ist das nicht vorhanden kann eine App entfernt werden, indem Sie in den Papierkorb gezogen wird. Dieser muss zu vollständigen Deinstallation dann geleert werden.

## Autostart

Innerhalb vom Autostart können sich beispielsweise Schad- oder Adware persistieren, bzw. hierdurch auch nach dem ausschlaten und ggf. löschen der infizierten Datei wieder erneut ausbreiten. Daher ist es sinnvoll auch hier sich zumindest mal anzuschauen was bzw. welche Programme beim Start des Betriebsystems alle auch gestartet werden.

### Windows

Für Windows 10/11 können Programme die automatisch mit Windows gestartet werden über den Taskmanager verwaltet werden. Im Taskmamager angelagt können über den Reiter (Windows 10 oben/11 rechte Seite) "Autostart von Apps" einzelne Programme über oben deaktivieren bzw. rechtsklick -> deaktivieren deaktiviert werden.

### Mac

Unter MacOS können Autostart-Programme über die Systemeinstellungen eingesehen werden. Über die Systemeinstellungen muss als nächstes die Einstellung Benutzer & Gruppen gewählt werden. Sind mehrere Nutzer vorhanden muss auf der linken Seite der jeweilige Benutzer bzw. die Benutzerin gewählt werden. auf der rechten Seite wird werden dann die Anwendungen angzeigt, die beim Start automatisch geöffnet werden. Von hier aus können Programme entfernt werden, indem Sie ausgewählt werden und auf das "-" geklickt wird.

## Browser-Addons

Jeder Browser bietet die Möglichkeit Addons an einer zentralen Stelle zu verwalten. Da die meisten Browsern auf Chrome bzw. Firefox aufbauen und Safari auf Desktop sowie Smartphones von Apple vorinstalliert ist konzentriert sich die Anleitung auf diese drei Browser.

### Chromium-basiert (Chrome, Edge, Brave, Opera)

Addons in Chrome(ium)basierten Browsern wie Edge, Opera und Brave sowie auch Google Chrome können über die drei Punkte bzw. das Burgermenü (oben rechts) Erweiterungen -> Erweiterungen verwalten überprüft und entfernt werden.

Über den Schieberegler können Addons deaktiviert werden. Sie bleiben aber im Browser enthalten und können bei bedarf wieder aktiviert werden.

Über "Entfernen" können diese aus dem Browser entfernt werden

### Firefox (und Forks wie Waterfox/Librewolf/Floorp)

In Firefox sind Addons über die Menüschaltfläche (Burger-Menü) -> Erweiterungen und Themes zu erreichen. Bei Klick hierauf gelangt man zu der Übersicht aller installierten Addons. Diese können über den Schieberegeler deaktiviert oder über die drei Punkte -> Entfernen gelöscht werden.
Themes können über den Reiter auf der linken Seite über die drei Punkte gelöscht werden.

### Safari

In Safari können Addons über Einstellungen -> Erweiterungen verwaltet werden.
Addons können in Safari über die Checkbox auf der linken Seite des Icons de- bzw. aktiviert werden.
Beim Klick auf das Icon bzw. das Addon wird unterhalb Zusammenfassung Deinstallationsbutton angzezeigt. Über diesen können einzelne Erweiterungen deinstalliert werden.

# Updates

Das Ziel ist es Updates möglichst schnell zu installieren. Das funktioniert am besten mit automatischen Updates. Ist dies nicht möglich sollten Updates schnellstmöglich über den vom Hersteller empfohlenen Weg installiert werden, dies ist häufig über die Seite des Entwicklers.

## Betriebssystem

Automatische Updates sind für das Betriebssystem Windows selbst als Standardeinstellung aktiviert. Manuell kann nach Updates über Einstellungen -> Updates und Sicherheit bzw. Windows Update gesucht werden. Ein Zugriff zu dieser Anzeige ist auch unter diesem Link möglich: [Windows Update](ms-settings:windowsupdate?activationSource=SMC-IA-4027667)

### Software aus dem Internet

#### Automatische Updates

Dies ist der Best-Case. Updates werden beim starten der App installiert. Beispiele hierfür sind Discord oder auch der Browser, der ein Update während einer Sitzung herunterlädt und beim Schließen installiert.

#### Halbautomatische Updates

Dies geschieht in der Regel über die Anwendung. Entweder sucht die App auch beim start nach einen Update und empfhielt über eine Meldung das Update zu installieren. Hier sollte der Text in der Regel akzeptiert werden und das Update installieren. Zu beachten ist hier jedoch, dass gerade bei Browser versucht wird diese Methode von Schadsoftware auszunutzen und solche Meldungen anzuzeigen jedoch um Schad- bzw. zumindest Adware zu installieren. Deshalb sollte bei unstimmigkeiten auf der Webseite des Entwicklers überprüft werden ob z. B. wirklich eine neue Version in passenden Zeitraum veröffentlich wurde.

#### Manuelle Updates

Hier wird in der Regel vom Anwender gefordert die App selbst zu aktualisieren. Es gibt hier nochmal eine Unterscheidung bei dem Herunterladen der neuen Version. Einige Anwendungen bieten in ihrer Software ein eigenen Downloader der die Anwendung aktualisert, bzw. den Prozess anstößt. Auf der anderen Seite gibt es auch Anwendung die zur Überprüfung auf die eigene Seite verweisen und dort eine neue Version bereitstellen.

Der Ort an dem diese Informationen bereitgestellt werden ist jedoch bei den meisten Anwendungen zumindest ähnlich.

Updates bzw. die aktuelle Version werden bei Anwendungen häufig unter einem Punkt wie Über / Hilfe / Info gezeigt. Dort kann auch, sofern vorhanden nach neuen Versionen gesucht und diese ggf. gleich installiert werden. Die Suche nach Updates wird häufig mit "Nach Updates/Aktualisierungen suchen" gestartet. Die aktuelle Version, die ggf. mit einer neuen Version vom Hersteller verglichen werden kann ist häufig unter Über zu finden. Leider ist dies nicht vollständig vereinheitlicht, jedoch sind diese zumindest ein Anhaltspunkt für das suchen solcher Optionen.

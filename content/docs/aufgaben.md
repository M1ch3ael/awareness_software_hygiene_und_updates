# Aufgaben

## 1. Hygiene
Erstelle eine Inventarliste über die installierten Apps auf deine Geräten.  Benutze folgendes Template.
Bei mindestens x Programmen sollte hinzugefügt werden was wann installiert wurde, für welchen Grund und wann diese zuletzt genutzt wurde.

Sofern nicht schon zuvor gemacht, gib an welche der Apps deinstalliert wurde.

Zudem sollten hier sofern vorhanden der Autostart protokolliert sowie die installierten Browseraddons aufgeschrieben werden. Alle die nicht benögigt werden können entfernt werden.


## 2. Software updates
In der Tabelle von zuvor tragen Sie nun bei denen Anwendungen die nicht deinstalliert wurden  mindestens 5  ein ob es hierfür ein Update gibt, welche Version ist aktuell installiert ist, ob es innerhalb dieser Zeit Schwachstellen gepacht wurde.
